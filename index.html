Archivo plano:

un archivo plano (también conocido como archivo de texto o archivo de datos) es un tipo de archivo que contiene datos en un formato de texto plano, sin formato binario o de formato especializado. Este tipo de archivo se compone de una serie de líneas de texto sin ningún tipo de formato específico, y cada línea puede contener una cantidad variable de caracteres y/o números separados por algún tipo de delimitador, como una coma, un tabulador, un espacio en blanco, entre otros.

ejemplo archivo plano:


				Juan Perez, 25, 1.75, 75
				Maria Gomez, 30, 1.68, 60
				Pedro Rodriguez, 40, 1.80, 85

				______________________________________________

				const fs = require('fs');

				fs.readFile('datos.txt', 'utf8', (err, data) => {
  				if (err) throw err;

  				const lineas = data.trim().split('\n');
 				for (const linea of lineas) {
    				const datos = linea.split(', ');
    				const nombre = datos[0];
    				const edad = parseInt(datos[1]);
    				const altura = parseFloat(datos[2]);
    				const peso = parseFloat(datos[3]);
    			console.log(nombre, edad, altura, peso);
  				}
				});

Archivo XML:

XML es un lenguaje de marcado XML desarrollado por Mozilla para crear interfaces de usuario para aplicaciones basadas en Mozilla como Firefox, Thunderbird y SeaMonkey. Un archivo XUL es un archivo de texto plano que utiliza la sintaxis XML para definir la estructura y el contenido de una interfaz de usuario. En un archivo XUL, se pueden definir elementos como ventanas, cuadros de diálogo, menús, barras de herramientas, botones, campos de texto y otros elementos de la interfaz de usuario.

Archivo tipo json:

JSON (JavaScript Object Notation) es un formato de intercambio de datos que utiliza una sintaxis sencilla y fácil de leer para representar objetos y datos estructurados. Un archivo JSON es un archivo de texto plano que utiliza la sintaxis JSON para almacenar datos estructurados en un formato legible para las computadoras.

archivo parquet

Parquet es un formato de archivo columnar utilizado para el almacenamiento y procesamiento eficiente de grandes conjuntos de datos en sistemas distribuidos. Se utiliza comúnmente en herramientas de procesamiento de datos como Apache Hadoop y Apache Spark. El formato Parquet almacena los datos en columnas en lugar de en filas, lo que lo hace más eficiente en términos de almacenamiento y procesamiento de datos. Además, utiliza técnicas de compresión avanzadas para reducir el tamaño del archivo y mejorar el rendimiento.

ejemplo archivo parquet:


					
						import pyarrow as pa
						import pyarrow.parquet as pq

						# Crear un esquema para el archivo Parquet
							schema = pa.schema([
   								("nombre", pa.string()),
  								("edad", pa.int32()),
    							("altura", pa.float64())
							])

						# Crear un conjunto de datos con algunos datos
							datos = [
   								 {"nombre": "Ana", "edad": 30, "altura": 1.65},
   								 {"nombre": "Juan", "edad": 25, "altura": 1.75},
 								 {"nombre": "María", "edad": 35, "altura": 1.80}
							]

						# Crear una tabla PyArrow a partir del esquema y los datos
							tabla = pa.Table.from_pydict({k: [d.get(k) for d in datos] for k in schema.names})

						# Escribir la tabla en un archivo Parquet
						pq.write_table(tabla, 'datos.parquet')
